# ArcFit - Frontend

## 🌐 Rotas Principais

- Landing page com informações da academia (`/`)

- Área Administrativa (`/admin`)

- Área Cliente (`/app`)

## 🛠️ Scripts Disponíveis

```bash
# Desenvolvimento com Turbopack
npm run dev

# Build de produção com Turbopack
npm run build

# Iniciar servidor de produção
npm start

# Análise de código
npm run lint
```
